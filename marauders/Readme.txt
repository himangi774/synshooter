CSE251 Sample OpenGL Code
~~~~~~~~~~~~~~~~~~~~~~~~~

How to run:
-> The makefile compiles the code. (Type 'make' on the terminal)
-> Now type './v2' (on terminal) to run the executable. {linux}

Controls:
-> You can use the arrow keys to move around 
-> Space shoots a laser
-> The lup and down arrow keys rotate the canon

Need to install openAl
	sudo apt-get install libalut-dev libopenal-dev
